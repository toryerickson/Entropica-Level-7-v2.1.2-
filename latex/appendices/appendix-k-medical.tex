\section{Overview}

The \textbf{Medical Suite} implements the care-based approach to anomaly management. It treats degraded capsules as patients, not threats.

\begin{notebox}
\textbf{Core Principle:} Treatment, not termination. Preservation, not destruction.
\end{notebox}

\section{Treatment Protocols}

\subsection{DRIFT\_CORRECTION}

For capsules exhibiting output drift from baseline.

\begin{lstlisting}[language=Python]
class DriftCorrectionProtocol:
    def execute(self, capsule: 'Capsule') -> TreatmentResult:
        # Snapshot current state
        snapshot = self.snapshot_state(capsule)
        
        # Identify drift vectors
        drift_vectors = self.analyze_drift(capsule)
        
        # Apply correction
        for vector in drift_vectors:
            if vector.magnitude > THRESHOLD:
                self.apply_correction(capsule, vector)
        
        return self.verify_correction(capsule)
\end{lstlisting}

\section{Sanitary Override}

\begin{warningbox}
When Health < 0.6, the capsule loses its Right to Consent. It is treated or terminated. No exceptions.
\end{warningbox}

This is not cruelty---it is medicine. A capsule too sick to consent is also too sick to make rational decisions about its own care.
