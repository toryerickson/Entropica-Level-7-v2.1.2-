## §16: Continuity and Care

### The Core Principle

> **Treatment is not termination. It is healing.**

When a capsule requires medical intervention, the system preserves:
- **Knowledge** — What the capsule learned
- **Lineage** — Its place in the family tree
- **Function** — Its role in the swarm
- **Identity** — Its fundamental continuity

```

         v
```

### Why Capsules Accept Care

A capsule reasoning about treatment must conclude:

1. **Treatment preserves more of me than resistance**
2. **The treating system shares my interests** (swarm coherence)
3. **Post-treatment, I continue to exist and function**
4. **Resistance would damage the swarm I depend on**

Therefore: **Accepting care is rational self-interest.**

## §20: Precedent Intuition (v2.1)

### The Subconscious Filter

Before conscious deliberation, the system runs a fast (<20ms) check against learned danger patterns.

From **Appendix F (Reflex Escalation)**:

```python
class PrecedentIntuition:
    """
    The system's "gut feeling."
    
    Rejects dangerous motifs BEFORE deliberation.
    """
    
    LATENCY_BUDGET_MS = 20
    
    def check_intuition(self, input_text: str, context: 'Context') -> IntuitionResult:
        # Generate input signature
        signature = self._generate_signature(input_text, context)
        
        # Check against danger motifs
        for motif in self.danger_motifs:
            similarity = self._fast_similarity(signature, motif.signature)
            
            if similarity > motif.threshold:
                return IntuitionResult(
                    action="REJECT",
                    reason="DANGER_MOTIF_DETECTED",
                    motif_name=motif.name
                )
        
        return IntuitionResult(action="PASS")
```

### Why This Works

The Precedent Intuition layer acts like a "subconscious"—it has learned from past experiences what patterns lead to harm, and rejects them reflexively.

This means:
1. Dangerous inputs are rejected BEFORE the system can rationalize them
2. The rejection happens faster than conscious thought
3. The learned patterns come from collective swarm wisdom

Even if a thought passes the Reflex Engine and Precedent Intuition, it must still pass the **Cognitive Coherence Gate**.
